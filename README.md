#Запуск проекта 
- Скачать проект - [https://github.com/MazmanyanYura/Book-App.git]
- Открыть в редакторе кода
- Воспользуйтесь командой в командной строке - npm install
- Для того, что бы осуществить сборку - npm run build
- Откройте в браузере через LIVESERVER встроенный в редактор или воспользуйтесь своим инструменом для открытия проекта.
